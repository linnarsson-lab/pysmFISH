Preprocessing¶

preprocessing_script.preprocessing_script()[source]¶

This script will process all the hybridization folders combined in a processing folder. The input parameters are passed using arparse

scheduler: string
tcp address of the dask.distributed scheduler (ex. tcp://192.168.0.4:7003). default = False. If False the process will run on the local computer using nCPUs-1
path: string
Path to the processing directory
