Welcome to pysmFISH!¶

pysmFISH is a library for the analysis of single molecule fluorescence hybridization (smFISH).

pysmFISH include an API and a pipeline built to analyse large datasets.

Warning

the documentation is still not fully complete and changes will be made frequently
